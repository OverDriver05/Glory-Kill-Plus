# **GLORY KILL +**
This is the primary repository for collaborative efforts between Doom developers on **Glory Kill +**. This is the bleeding-edge version that is constantly being developed on, and not meant in any way shape or form to be representative of the final version of the mod.

# **WHAT IS GLORY KILL +?**
Similar to how [**BDLite**](https://teamouse.net/games/doom/bdlite/) is to [**Brutal DOOM**](https://www.moddb.com/mods/brutal-doom), **GLORY KILL +** aims to bring the most-memorable mechanic from the modern _DOOM_ games to the classic games, all while being lighweight and is specificall made with the modders in mind.
